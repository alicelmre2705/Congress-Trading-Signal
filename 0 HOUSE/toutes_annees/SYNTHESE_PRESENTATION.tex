\subsection*{Audit de la passe LLM ticker (\file{\_ocr\_echantillon\_diff.csv})}

Sur les \textbf{894 tickers LLM}, tous rapprochables d'un bioguide Quiver~:
\textbf{743 concordants = 83,2\,\%} (pilote 2025~: 93\,\%).
L'écart de 10~pts s'explique par les membres absents de Quiver (trusts familiaux Harshbarger)
et les tickers delistés (XLNX absorbé par AMD\,2022, CTL rebrandé LUMN)~---
\textbf{aucun faux ticker LLM détecté}.

sur les scans anciens/tournés (modes~: confusion~4$\leftrightarrow$1, date répliquée).
\textbf{2025--2026 = propres}~: prec.~date 0,54--0,77.}

\section{Limites connues}
% =====================================================================

\begin{itemize}
  \item \textbf{OCR = échantillon, pas exhaustif.}
        Les 547~scannés ne sont \emph{pas} tous extraits. Livrable OCR = 70~docs pour
        \emph{mesurer} la qualité, pas fournir tous les trades papier.
  \item \textbf{Cluster~C non vérifiable.}
        Quiver couvre quasi-aucun petit déposant manuscrit~: prec.~date 0,01 reflète
        l'absence de vérité-terrain, pas un échec OCR.
  \item \textbf{Dates résiduelles (2021--2024).}
        Deux modes persistants après garde-fou~: confusion 4$\leftrightarrow$1 sur doc
        tourné (année lue à --3~ans)~; date copiée sur toutes les lignes. Non corrigés
        car correction heuristique masquerait des erreurs réelles.
  \item \textbf{Tickers non-cotés laissés vides.}
        Munis, obligations, options, trusts privés~: \texttt{ticker}~null à dessein,
        jamais deviné.
\end{itemize}
